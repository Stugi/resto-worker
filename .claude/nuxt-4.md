# UI Components (Shadcn)

- Мы используем `shadcn-nuxt`.
- Компоненты лежат в `app/components/ui`.
- Все компоненты доступны через АВТОИМПОРТ.
- НЕ ПИШИ `import { Button } from '...'`, просто используй `<Button>` в шаблоне.
- Если нужного компонента нет в `app/components/ui`, сообщи мне, что его нужно установить через `yarn dlx shadcn-vue@latest add [component]`.
